# Formulas

## Summation

$1 + 2 + ... + k = \frac{1}{2}k(k + 1)$

$1^2 + 2^2 + ··· + k^2 = \frac{1}{6} k(k + 1)(2k + 1)$

$\sum^{n}_{i=1} (\alpha \times u_i + v_i)^2 = \alpha^2\sum^{n}_{i=1} |u_i|^2 + 2\alpha\sum^{n}_{i=1}u_iv_i + \sum^{n}_{i=1}|v_i|^2$

$|\sum^{n}_{j=1}a_{mj}v_j| = \sum^{n}_{j=1}|a_{jm}||v_j|$
